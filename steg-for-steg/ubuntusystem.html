Sette opp en Ubuntu Server

Nå som vi har laget den virtuelle maskinen må vi ordne systemet.

Steg 1: Åpne VMen

Klikk på den virtuelle maskinen for å åpne den, og deretter på "Start".

Steg 2: Install Ubuntu Server

Velg "Try or Install Ubuntu Server" for å starte installasjonsprosessen.

Steg 3: Konfigurer systemet

1. Velg engelsk som språk. Dette anbefales for servere.


2. Velg riktig tastaturoppsett for ditt tastatur, og klikk deretter "Done".


3. Velg det første alternativet, "Ubuntu Server, og gå videre.


4. Du kan hoppe over Network configuration, Proxy og Archive mirror dersom serveren kun skal brukes lokalt. Klikk "Done" til du kommer til Guided storage configuration.


5. Velg "Use an entire disk" og huk av for "Set up this disk as an LVM group".


6. Fortsett med "Done" og "Continue" for å starte installasjonsprosessen.


7. For profil konfigurasjon, velg informasjon som er lett å huske.


8. Hopp over Ubuntu Pro og fortsett.


9. Installer OpenSSh Server og gå videre til "Reboot Now" kommer. Velg deretter "Reboot now".

Nå er den satt opp og klar til bruk!

Det neste blir å sette opp en SSH kobling for lettere tilgang.
