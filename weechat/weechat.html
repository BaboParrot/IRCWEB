Weechat installasjon

Steg 1: Oppdater pakkene

Kjør følgende kommando for å oppdatere pakkene:

sudo apt update

Steg 2: Last ned Weechat

Kjør følgende kommando for å installere Weechat:

sudo apt install weechat

(VALGFRITT) Sjekk om installasjonen fungerer ved å skrive:

weechat

For å avslutte, skriv /quit eller trykk Ctrl + D.

Klar for bruk

Nå har du Weechat klar til bruk!
